# HR Attrition Analysis

## Project Overview
This project analyzes employee data to understand factors affecting attrition and predict whether an employee will leave the company.

## Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- Machine Learning Models
- Power BI Dashboard

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Results
Random Forest achieved the highest accuracy (~82%) for predicting employee attrition.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Model Building
- Interactive Power BI Dashboard

## Files
- attrition_analysis.ipynb → ML work
- dataset.csv → dataset
- dashboard.pbix → Power BI dashboard

## Conclusion
The project helps identify key factors influencing employee attrition and supports data-driven decision making.
